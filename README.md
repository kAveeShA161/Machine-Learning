# 🤖 Supervised Machine Learning Cookbook

This repository contains a curated collection of Jupyter Notebooks that break down core **Supervised Machine Learning algorithms**, data preprocessing methodologies, and model evaluation metrics. 

This repository serves as a personal study guide, theoretical reference, and hands-on laboratory for understanding both the mathematical execution and practical implementation of modern machine learning workflows.

---

## 🛠️ Tech Stack & Dependencies

* **Language:** Python
* **Environment:** Jupyter Notebook
* **Core Libraries:**
  * `scikit-learn` (Algorithm implementations & evaluation)
  * `pandas` (Data manipulation & analysis)
  * `numpy` (Numerical computation & matrix operations)
  * `matplotlib` (Data visualization & performance curves)

---

## 📁 Repository Structure

The project is organized into standalone, modular directories focusing on specific algorithms, preprocessing techniques, or evaluation steps:

```plaintext
├── Linear Regression Model/       # Single & multiple variable regression notebooks + datasets
├── LogisticRegression/            # Binary classification with logistic regression
├── Decision Tree/                 # Decision tree classifier implementation
├── Random Forest/                 # Ensemble tree-based classification and bagging
├── SupportVectorMachine/          # SVM classifier (linear and kernel tricks)
├── NaiveBayes/                    # Probabilistic Naive Bayes classifier
├── K-Nearest Neighbors/           # Distance-based KNN classifier implementation
│
├── Feature Scaling/               # Data normalization and standardization techniques
├── OneHotEncoding/                # Categorical variable dummy encoding
├── Train Test Split/              # Train-test data partitioning methodology
│
├── Cross Validation Score/        # K-fold cross-validation evaluation techniques
├── Confusion Matrix/              # Classification performance metrics (Precision, Recall, F1)
├── Regression Evaluation Models/  # Regression performance metrics (MSE, RMSE, R²)
└── Hyperparameter Tuning/         # Grid Search and Random Search for model optimization
```
